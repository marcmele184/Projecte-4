**Fase 2: Treball per parelles**

Treballant per parelles:

1\.       Discussió i Consens: Comparen les seves respostes individuals (Fase 1).

2\.     Elaboració d'una Proposta Unificada: Heu de consensuar i dissenyar el vostre propi Esquema 3-2-1 de Còpies (3 còpies, 2 mitjans, 1 fora de lloc) basat en els requisits del cas.

| Element | Proposta | Per què? |
| :---- | :---- | :---- |
|  |  |  |
| **Dades Crítiques** | Bases de dades de comptabilitat i clients, documents de projectes i carpetes personals dels usuaris | Són les dades més importants per a l’empresa. Les bases de dades són molt importants i els arxius de la carpeta *Documents* dels tècnics també tenen informació important. |
| **Quan fer còpies** | Còpia completa un cop per setmana (diumenge a la nit) \+ còpies incrementals cada 4 hores entre setmana | Així, si passa alguna cosa, només es perdran com a màxim 4 hores de treball. La còpia setmanal completa és el punt principal per  recuperar les dades. |
| **Tipus de còpia** | Completa setmanal \+ Incremental entre setmana | La còpia completa guarda tot, i les incrementals només el que ha canviat, per poder estalviar espai i temps. |
| **Mitjà 1 (Local)** | NAS dins de l’empresa | Permet recuperar les dades d’una forma més ràpida en cas que hi hagi algun problema. |
| **Mitjà 2 (Extern)** | Núvol (Cloud) | Permet tenir una còpia fora de l’oficina per protegir-se de desastres físics com incendis. |
| **Mitjà 3 (Extra)** | Disc dur extern fora de l’oficina | Per completar la norma 3-2-1 en cas que hi hagi algun problema a l’empresa i no puguem accedir al núvol o al NAS. |
